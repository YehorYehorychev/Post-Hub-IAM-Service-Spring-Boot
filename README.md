# IAM SERVICE [Spring Boot Project]

This is a Spring Boot project that implements an Identity and Access Management (IAM) service. The service provides functionalities for user authentication, authorization, and management of user roles and permissions.

## Features
- User Registration and Login
- Role-Based Access Control (RBAC)
- JWT Token Generation and Validation
- Password Encryption
- User Profile Management
- RESTful API Endpoints

## Technologies Used
- Spring Boot
- Spring Security
- JWT (JSON Web Tokens)
- Spring Data JPA
